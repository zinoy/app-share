Javascript

可在页面中定制微信，手机QQ，QQ空间APP内的分享内容。官方说明文档

由于手机QQ限制，分享URL必须与当前页面URL在同一域名，否则以下设置将不会生效。

方法一

setShareInfo({ title: __shareDef.title, // 分享标题 summary: __shareDef.desc, // 分享内容 pic: __shareDef.img, // 分享图片 url: __shareDef.url, // 分享链接 });

或者，使用方法二（此方法同时兼容微信APP，可省略微信的配置代码）

//首先，同微信的配置方式一样，需获取微信的签名。注：别克微信分享地址为：http://www.buick.com.cn/weixinjssign/getbuicksign.aspx $.post("http://www.chevrolet.com.cn/chevy-api/getsign.aspx", {signurl: window.location.href.replace(/#.*$/ig, "")}, function(config){ config.swapTitleInWX = true;// 朋友圈中是否将标题和内容对调（根据实际需要修改） setShareInfo({ title: __shareDef.title, // 分享标题 summary: __shareDef.desc, // 分享内容 pic: __shareDef.img, // 分享图片 url: __shareDef.url, // 分享链接 // 微信权限验证配置信息，若不在微信传播，可忽略 WXconfig: config }); }, "json");
APP判断脚本
var isQzone = !!window.mqq; // or var isQzone = window.navigator.userAgent.indexOf("Qzone/") !== -1; var isQQ = !!window.navigator.userAgent.match(/QQ\/([\d\.]+)/);
