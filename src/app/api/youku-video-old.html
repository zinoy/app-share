API 说明

上汽通用别克&雪佛兰网站上使用的视频, 基本上都是存在放在youku服务器上, 视频由客服人员上传到youku服务器上, 并把视频加入Youku白名单中(视频上无广告与youku水印logo). 再通过youku提供的api接口, 可获取动态视频地址.
通过调用以下接口地址, 可直接得到youku视频地址:

  • Buick: //www.buick.com.cn/getvideo/youku/getvideo.ashx?iswebserice=1&vid=xxxxxxxx
  • Chevy: //www.chevrolet.com.cn/chevy-api/getvideo.ashx?iswebserice=1&vid=xxxxxxx
注:
1. 当参数不正确或视频不存在，则会显示雪花底的404视频。
2. 对于PC网站, 为了向下兼容IE8浏览器, 建议结合VideoJS一起使用。

参数说明:
  • vid : youku视频的vid, 一般Account会提供vid, 如:XOTI5NzMwMjky。
  • type : [可选, 默认值为mp4] 视频类型, 可取值为:mp4, flv, hd2等。
  • iswebserice : [可选, 默认值为0] 是否为Youku webservice接口, 1:是, 0:否。*1
  • year : [可选]年份, 如果iswebserice=1, 则不用传year这参数. 如:2015。
*1: 从2016年开始, youku启用webservice接口方式获取视频地址, 为了向下兼容2015年视频接口, 所以增加iswebserice=1参数, 用于区别新旧youku视频接口。新youku视频接口, 不再需要传递year参数。
