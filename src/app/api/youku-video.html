API 说明

上汽通用别克&雪佛兰网站上使用的视频, 基本上都是存在放在Youku服务器上, 视频由客服人员上传到Youku服务器上, 并把视频加入Youku白名单中(视频上无广告与Youku水印logo). 再通过youku提供的api接口, 可获取动态视频地址.
从2017年2月28日开始，Youku视频已不能通过之前动态程序调用返回mp4 URL地址了，现改为调用Youku视频播放器来播放视频。

注:
1. 优点：使用方法更简单了，且不会再占用服务器资源。
2. 缺点：使用受限，播放器样式不能再定义，还有些bugs。

为了方便调用，已把youku播放器的API方法封装在testdrive.js里，引入testdrive.js就可以了。

参数说明:
  • vid : youku视频的vid, 一般Account会提供vid, 如:XOTI5NzMwMjky。
  • poster : [可选, 默认值为空] 定义视频预览图，在PC上使用swf播放器不支持此属性，手机支持。
  • autoplay : [可选, 默认值为0] 手机不支持此属性。
  • controls : [可选，默认值为true]手机不支持此属性。
  • rebuild : [可选，默认值为false]就当前div标签再重新再生成新的视频。
  • preview : [可选]显示预览图。因为swf播放器不支持poster属性，所以通过此方法来显示预览图，当点击预览图时，才会加载Youku swf播放器。此属性需要配合autoplay=true一起使用。在手机上不需要定义preview属性，否则会产生点击两次才会播放视频。
  • onPlayerReady : [可选]Function 当播放器初始化完成时。
  • onPlayStart : [可选]Function 当视频开始播放时。只会执行一次，暂停再播放时不会产生此事件。
  • onPlayEnd : [可选]Function 当视频播放结束时。
*: 因为只能使用Youku视频播放器来播放Youku网站上的视频，所以video标签很多功能不能实现，如果Youku播放器无法实现网站上的功能，请与客服人员说明情况，并把视频放在自己服务器上。

//基本调用方式，具体查看下方的Demo var player = hylink.videoPlayer("your_div_id",{vid:"XOTI5NzMwMjky"}); player.playVideo();//播放视频，此方法需要在视频onPlayerReady后方可执行 player.pauseVideo();//播放视频，此方法需要在视频onPlayerReady后方可执行
