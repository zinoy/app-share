API 说明

上汽通用别克服务器获取当前服务器时间戳。

参数说明:
  • format : 当前可用的值为js，默认是直接显示当前服务器时间戳。

回调返回参数说明:
  • 默认返回 : 时间戳文本，如：1489396722342。
  • format=js : 返回如：var CURRENT_SYSTEM_TIME = new Date(1489396722342);

方式A（一次获取服务器时间，推荐）：

<script src="//www.buick.com.cn/apinew/timespan.aspx?format=js"></script> <!--返回var CURRENT_SYSTEM_TIME = new Date(1489396722342);，可直接使用CURRENT_SYSTEM_TIME变量-->

方式B（可多次请求获取服务器时间）：

//通过ajax方式去获取时间。 $.get("//www.buick.com.cn/apinew/timespan.aspx").done(function(t){ console.log(new Date(parseInt(t))); }).fail(function(){console.error("timespan.aspx Error!")});
